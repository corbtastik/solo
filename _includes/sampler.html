Yolo Sampler

The Yolo single page sampler platter, each color palette is bundled in `_sass/yolo/themes`. Feel free to add your own.

  • default: The default style configured in _config.yml.
  • arcade: 80s video game arcade ambiance.
  • blue: Just blue.
  • corbs: My personal styleo (set as default).
  • dark: Token dark styleo.
  • dracula: Token dark styleo.
  • eightyfour: 80s centric colors.
  • githubdark: Github dark colors.
  • green: Just green.
  • light: Token light styleo.
  • monokai: Classic Sublime Text colors.
  • nightowl: Colors for the night owls out there.
  • purple: Skeletor would be happy with this styleo.
  • red: Just red.
  • soft: Soft like cotton theme.
  • solarized: Classic solarized dark colors.
